WordPress Activity Dashboard

Todo el control de tu sitio en un solo lugar. Recolección de datos inteligente y visualización en tiempo real.

¿Qué es WAD?

WAD es un plugin de analítica para WordPress que centraliza métricas de contenido, interacciones, usuarios y plugins. Su arquitectura modular permite un seguimiento preciso sin sobrecargar tu base de datos.

  • Arquitectura: Basado en clases PHP (`WAD_Data`) para un manejo eficiente de datos.
  • Visualización: Gráficos interactivos mediante Chart.js.
  • Performance: Encolado condicional de activos (solo carga donde se necesita).
